Reflexivity:
A relation is reflexive if, for every element of the domain, there is a column where both rows have the same value.

Symmetry:
For every column with rows x and y, there is another column with rows y and x.

Transitivity:
If there is a column with rows x and y, and a column with rows y and z, then there is also a column with rows x and z.
